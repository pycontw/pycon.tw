活動簡介

兩場 Sprint 活動安排在會議三天以外的時間。當天的 Sprint 將有開源專案的 負責人，帶著他們專案待解決的問題、issue 來現場與大家分享與解說，參加的 人可以就自己感興趣的專案一同加入。透過這個互動機會將能更了解自己關注的 相關專案領域。歡迎帶自己的開源專案來找同好。對已經有一定 Python 基礎的 人，想參加開源專案開放但不知如何加入，或者害羞怕自己不了解該專案環境的 人，參加 Sprint 是個大展身手的好機會！

對於沒有把握自己是否能加入開發的朋友，四月的活動也會有 workshop 形式的 工作坊，歡迎一起來玩！

2016-04-16 (六): Python Workshop: Tutorials and Projects On PyCon Taiwan by PyLadies Taiwan * Taipei.py

結合 Sprints 與 Tutorials 的活動，由 PyLadies Taiwan 與 Taipei.py 協辦。

詳細活動內容請見報名網頁

  • Projects On (Sprint): http://www.meetup.com/Taipei-py/events/228997805/
  • Tutorials:
    • PyLadies (限 PyLadies) http://www.meetup.com/PyLadiesTW/events/229629711/
    • Taipei.py http://www.meetup.com/Taipei-py/events/229608148/

2016-06-11 (六): PyCon Taiwan Sprint Day

大會後的正式 Sprint 活動，報名預計在大會報名結束後開放。 屆時會在 Python Taiwan 郵件列表、Twitter，以及 Facebook 上發佈活動訊息。

場地

並不是在大會場地中央研究院內，而是在大安區的 CLBC 共同辦公室。

  • CLBC 4F (Map)
  • 台北市大安區復興南路一段293 號 4 樓

入口在巷子裡，肉羹與牛肉麵店中間的公寓門，門鈴貼有「CLBC」的標籤 （街景）
