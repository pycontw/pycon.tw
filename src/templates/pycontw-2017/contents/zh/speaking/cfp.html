投稿募集（Call for Proposals）

臺灣 PyCon 2017 正式開始徵稿，接受包括議程 (Talk) 與課程 (Tutorial) 的稿件喔！

重要日期

  • 議程 ＆ 課程 開放徵稿：02月15日
  • 課程投稿截止：03月15日
  • 議程投稿截止：03月15日
  • 公告完整議程：04月30日

因為 PyCon 的目的之一在於推廣 Python 程式語言與其在教育上的影響力，您的議程（talk）預設將會被錄影、編輯。之後 PyCon Taiwan 會把錄影編輯後上傳至 pyvideo.org 該網站。如果不希望您的議程被記錄，請確定在送出投稿主題前，確認在「記錄選項（recording option）」中選擇正確的選項。更多資訊請參考錄影釋出。

特別要注意的事，每一位講者也需要購買會議的門票。（更多資訊請參考 Everyone Pays Policy）

如何提交您的稿件

您需要先在 tw.pycon.org 註冊一個新的帳號。在啟用您的帳號之後，就可以填寫講者資訊並且從您的控制面板中創建新的投搞。

我們強烈建議您儘早送出您的投稿。

稿件提交指南

議程（Talk）

我們接受泛 Python 相關的各類投稿，包括學術報告、商用專案或者案例研究等。正在閱讀本文件的您，想必也應該是要來投稿的！

今年 PyCon 臺灣 2017 的主題為「定義未來」，因此我們鼓勵講師給予您的 Python 應用、使用 Python 解決問題的的經驗， 或如何主持您的社群圈。我們也非常歡迎適合進階者內容的投稿。

絕大部分的議程時間長度為 30 分鐘。此外我們也安排了時間長度為 45 分鐘的議程，此類的議程將會是我們認為比較重要並且能夠極大化影響族群並給予顯著幫助的主題。需要注意的是，議程的時間長度將會包含開場與 Q&A 的時間。

議程可以使用的語言為：英文、中文、台語。

為了幫助講者組織他們的想法，我們重新設計了投稿表單。更多資訊請參考議程投稿教學。

非正式會議（Unconference）

本次我們將在會議的最後一天開一軌 Talks 作為非正式會議議程（Unconference Talk），目的是希望讓講者與會眾能有不一樣的互動。非正式會議的議程不僅僅歡迎 Python 相關的 proposal，我們也歡迎與 Python 非相關的題目。

一個非正式會議議程可以採用任何方式進行，講者可以自由發揮的使用該空間與時間，且大會對於非正式會議不做任何錄影和記錄，歡迎參與這個比起正規議程來得更刺激性的議題討論。

非正式會議的議程長度為 20 minutes 因此我們希望內容儘量是一個心得、經驗的分享，也請注意 20 minutes 包含與會眾的 Q&A 時間。系統會在會議前 10 天開放系統報名非正式會議議程，其議程則由會眾投票決定。

非正式會議的議程可以使用中文、英文或是台語。投稿格式請參見如何投稿非正式會議？頁面。

教學（Tutorial）

同演講，我們對教學的主題並不做任何限制。教學為半天或一整天的活動，相當於 3 或 6 小時的教學。投稿教學的方針基於演講，所以請先閱讀如何投稿演講？除了該文中的事項外，我們對教學投稿本身有額外的規範，請參見如何投稿教學？一文。
