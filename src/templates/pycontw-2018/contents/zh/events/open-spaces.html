開放空間

開放空間（Open Spaces）是自助式、聚會形式的活動，並且將會和大會議程平行進行。開放空間將會舉辦在會場的交誼廳（R4）。開放空間有別於大會大部分的會議內容，開放空間的議程規劃是由 PyCon 與會者「當場」計畫的。開放空間讓你能用任何你喜歡的方式定義、組織、規劃你自己的小聚。

對於曾經參加過過往 PyCon Taiwan 或是其他研討會的會眾，開放空間的概念也許讓你覺得似曾相識。沒有錯，開放空間基本上很類似 BoF，不過「開放空間 / Open Spaces」對於第一次參加的會眾來說是個更好猜測的詞語。

如何參與一個開放空間？

很簡單：人到場就好了 :)

在大會期間，報到櫃檯附近會有一個開放空間看板。看板即為開放空間的時間表，這樣你就可以找到你有興趣的開放空間。如果上面沒有你想討論的主題，那就從看板旁邊拿一張便利貼，寫下你想討論的主題，找一個有空位的時間貼上去吧！

有哪些開放空間？

開放空間有各種與會者想討論的主題。由於與會者們是非常多元化的，開放空間也是非常多元化的。開放空間的主題並沒有限制，可以是關於美甲、女權主義、瑜珈、甚至是桌遊。也可以是討論各種常見的技術議題，從資訊安全到你最喜歡的 Python 專案，到一些特定職位的討論，比如說 DevOps。

開放空間在哪裡？在什麼時候？

開放空間將會在大會兩天期間（星期五和星期六）舉辦在「R4」交誼廳。由於 R4 的場地不小，每個時段同時可以有三個開放空間活動在該場地進行。不過某些時段交誼廳會被做其他用途，實際的舉辦時間和主題以現場的看板為主。

如何舉辦一個開放空間？

在開放空間看板旁邊，我們會提供便利貼，這些便利貼可以用來表示你想舉辦開放空間。只需要在便利貼上寫下開放空間的名稱和簡介，並黏在看板上的空閒時段即可。如果可以附上一些你的基本聯絡資訊，比如說 Twitter ID、Gitter/GitHub ID、Telegram ID 等等，這樣也能幫助有興趣的會眾能夠順利聯絡上您並且進行交流。

開放空間的好主意

以下有幾個可以辦成開放空間的點子，提供您參考：

  • 用 Python 的 Maker 專案（例如樹莓派、IoT、居家自動化、機器人、四軸飛行器）
  • Hacking / Networking / DevOps
  • 資料視覺化、科學
  • 自然語言處理（例如聊天機器人）
  • 自我量化
  • 女性主義
  • 主辦人們的聚會：研討會、工作坊、在地社群的主辦人們
  • 網頁框架，例如 Django 或 Flask
  • 如何做開源貢獻：給初心者的協助、指導
  • Git
  • 如何避免職業倦怠
  • 求職相關討論

您也可以參考看看 PyCon US 2018 的 Open Spaces 活動，有許多很棒的開放空間的主題可以參考！
